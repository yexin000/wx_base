平台须知

        为传承和发扬文化产业，我们给从事文化产业的个人、企业、社会团体提供一个公平交易、交流的艺术品电商平台。

        平台提供国内外优质文化资源，文化相关信息，提供优质商户，优质民藏艺术品，以及国内外优质客户资源，以及宣传，广告平台内的商家和艺术品。如有交易，交流请双方在平台内自行完成。

        平台内入驻商户都是全国古玩艺术品从业精英，以精美的古代艺术品、现代艺术品、工艺品、非物质文化遗产为主。

        平台内商户禁止收售国家规定禁止的物品。如保护动物牙齿、皮毛、爆炸物等。上传者对自己发布的物品认真筛选，平台不承担任何责任。后果自负。如有发现平台会做出相应的惩罚。

        平台专场拍卖商户是经过认真筛选全国业内信誉良好，有责任心，有眼力的实业收藏家、实体店、社会团体、个人组成。

竞拍规则

1.平台给买家提供免费第三方担保交易。

2.平拍品上传、加价幅度、起拍价由商户自行定制。

3.竞拍成功后3日内买方付款确认，保持良好的信誉。

4.双方发货买卖双方保价发货，以免发生争议。如有发生自行协商解决。

5.其他问题请联系客服。
